# @vanduo-oss/vd3

The **Vue3-only line** of the Vanduo design system: one standalone package
that ships its own DTCG tokens, CSS tree, and typed `Vd*` components and
composables. Sole peer: `vue >=3.3` (no pinia).

## Install

Not yet published — pre-release scaffold. Once released:

```sh
pnpm add @vanduo-oss/vd3
```

## Status

Phase 0 scaffold only. The token/CSS foundation
(`vd3-token-css-foundation`) and the component/composable carryover
(`vd3-carryover`) are upcoming changes; full API documentation lands in
this file before the first release.
